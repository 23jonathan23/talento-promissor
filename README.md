Projeto Talento Promissor, é uma plataforma para trabalhos freelancers voltada a jovens universitários, que querem se desenvolver em sua área e ainda ser remunerado por isso.

## Tecnologias usadas: 
- React Hooks
- HTML
- CSS
- Adobe XD para prototipação
